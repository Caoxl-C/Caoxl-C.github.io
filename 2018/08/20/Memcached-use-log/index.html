关于服务启动选项

  • -d 是启动一个守护进程
  • -m 是分配给Memcache使用的内存数量，单位是MB
  • -u 是运行Memcache的用户
  • -l 是监听的服务器IP地址，可以有多个地址
  • -p 是设置Memcache监听的端口，最好是 1024 以上的端口
  • -c 是最大运行的并发连接数，默认是1024
  • -P 是设置保存 Memcache 的 pid 文件
