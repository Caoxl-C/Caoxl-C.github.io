CORS

CORS是一个W3C标准，全称是”跨域资源共享”（Cross-origin resource sharing）。

如果是客户端是浏览器，则在调用 API 的时候，基本上无法避免跨域问题。

不过处理也很简单，需要浏览器和服务器同时配合：浏览器在发起 OPTIONS preflight 请求的时候，服务器返回认可的一些信息就行了。
