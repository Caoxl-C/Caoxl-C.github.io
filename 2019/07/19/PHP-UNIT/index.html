PHPUnit

2019-07-19 PHP 阅读量

单元测试(Unit)是针对你的代码中非常少，而且相对独立的一部分代码来进行的测试。实际上，大部分单元测试都是针对单个方法进行的。
功能测试(Feature)是针对大面积代码进行的测试，包括多个对象之间的交互，甚至是对 JSON 端点的完整 HTTP 请求。

快速入门

环境

  • PHP 7.2
  • Laravel 5.8

定义并运行测试

  • Feature
  • Unit

可用断言

asseertCookie

断言响应中包含给定的 cookie：

asseertCookieExpired

断言响应中包含了给定的 cookie 且它已过期：

asseertCookieMissing

断言响应中不包含给定的 cookie：

asseertDontSee

断言响应中不包含给定的字符串：

asseertDontSeeText

断言给定字符串不包含在响应文本中：

asseertExactJson

断言响应中包含的数据与给定的 JSON 数据完全匹配：

asseertForbidden

断言响应中有禁止状态码：

asseertHeader

断言响应中有给定的包头：

asseertHeaderMissing

断言响应中没有给定的报头：

asseertJson

断言响应包含给定的 JSON 数据：

asseertJsonCount

断言响应 JSON 中有一个数组，其中包含给定键的预期元素数量：

  • asserJsonFragment

断言响应包含给定 JSON 片段：

asseertJsonMissing

断言响应未包含给定的 JSON 片段：

asseertJsonMissingExact

断言响应不包含确切的 JSON 片段：

asseertJsonMissingValidationErrors

断言响应没有给定键的 JSON 验证错误：

asseertJsonStructure

断言响应具有给定的 JSON 结构：

asseertJsonValidationErrors

断言响应具有给定键的给定 JSON 验证错误：

asseertLocation

断言响应在 Location 头部中具有给定的 URI 值：

asseertNotFound

断言响应具有未找到状态码：

asseertOk

断言响应有 200 状态码：

asseertPlainCookie

断言响应包含给定的 cookie （未加密）:

asseertRedirect

断言响应会重定向到给定的 URI：

asseertSee

断言给定的字符串包含在响应中：

asseertSeeInOrder

断言响应中有序包含了给定的字符串：

  • assertSeeText

断言给定的字符串包含在响应文本中：

asseertSeeTextInOrder

断言给定的字符串有序包含在响应文本中：

asseertSessionHas

断言 session 中包含给定的数据：

asseertSessionHasAll

断言 session 中有给定值列表：

asseertSessionHasErrors

断言 session 中包含一个给定字段的错误：

asseertSessionHasErrorsIn

断言 session 中具有给定的错误：

asseertSessionHasNoErrors

断言 session 没有错误：

asseertSessionDoesntHaveErrors

断言 session 没有给定键错误：

asseertSessionMissing

断言 session 中不包含给定键：

asseertStatus

断言响应中具有给定的状态码：

asseertSuccessful

断言响应中有成功的状态码：

asseertViewHas

断言响应视图是一段给定的数据：

asseertViewHasAll

断言响应视图具有给定的数据列表：

asseertViewIs

断言路由返回给定的视图：

asseertViewMissing

断言响应视图缺少一段绑定数据：

asseertAuthenticated

断言此用户已被认证。

asseertGuest

断言此用户未被认证。

asseertAuthenticatedAs

断言给定的用户被认证。

asseertCreadentials

断言给定的凭证有效。

asseertInvalidaCredentials

断言给定的凭证无效。

asseertDatabaseHas

断言数据库表中包含给定的数据。

asseertDatabaseMissing

断言数据库表中不包含给定的数据。

asseertSoftDeleted

断言数据库中的指定记录已软删除。
